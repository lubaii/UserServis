# Как остановить Spring Boot приложение

## Способ 1: По PID процесса (рекомендуется)

Найден процесс с PID: **83888**

```bash
# Завершить процесс
kill 83888

# Или принудительно, если не останавливается
kill -9 83888
```

## Способ 2: По порту (8080)

```bash
# Найти процесс по порту
lsof -i :8080
# или
netstat -tlnp | grep :8080
# или
ss -tlnp | grep :8080

# Завершить процесс по порту
fuser -k 8080/tcp
# или
kill $(lsof -t -i:8080)
```

## Способ 3: Поиск всех процессов Spring Boot

```bash
# Найти все процессы Spring Boot
ps aux | grep "spring-boot:run" | grep -v grep

# Найти все Java процессы приложения
ps aux | grep "com.userservice.Main" | grep -v grep

# Завершить все процессы Spring Boot
pkill -f "spring-boot:run"
```

## Способ 4: Через Maven (если запущено через mvn)

Если приложение запущено через `mvn spring-boot:run`, можно остановить через:
- `Ctrl+C` в терминале, где запущено приложение
- Или найти процесс Maven и завершить его

## Быстрая команда для остановки

```bash
# Остановить приложение на порту 8080
kill $(lsof -t -i:8080) 2>/dev/null || pkill -f "com.userservice.Main"
```

## Проверка, что приложение остановлено

```bash
# Проверить, что порт 8080 свободен
lsof -i :8080

# Проверить через curl
curl http://localhost:8080/actuator/health
# Должна быть ошибка подключения
```
